# bugos Delivery Drafts

Status: draft for human review

This directory contains delivery-support documents for prepare-only `bugos` services.

## Files

- `sample_buyer_handoff.md`: sample delivery checklist and acceptance language for prepare-only work.

## Boundaries

These documents do not create:

- a contract
- a privacy policy
- legal advice
- tax advice
- payment request
- invoice
- authorization to test targets
- authorization to submit reports
- authorization for outreach automation
- authorization for payment automation

## Required Review Before External Use

Before external use, review:

- legal wording
- privacy wording
- tax and invoicing requirements
- consumer/business distinction
- liability boundaries
- refund and delivery terms
- data-handling process

## Operating Rule

Use delivery drafts only for safe, prepare-only work. Do not use them to imply testing, scanning, exploitation, payload generation, credential handling, target interaction, or automatic submission.
